Что необходимо для регистрации ООО?

  • 1 Наименование на русском языке

    Дополнительно можно добавить на любом иностранном языке

  • 2 Адрес места нахождения (Юридический адрес)

    Адресом места нахождения ООО является адрес, по которому находится его исполнительный орган -- директор (генеральный директор). По действующему законодательству это должно быть нежилое помещение (офис).

  • 3 Виды деятельности ООО
    • подбираем по Классификатору ОКВЭД:
    • первый код ОКВЭД — основной
  • 4 Учредители и руководитель ООО

    Для физических лиц граждан РФ:

    • копия паспорта (фото + прописка)
    • личный номер ИНН

    Для физических лиц иностранных граждан:

    • копия паспорта с нотариально удостоверенным переводом на русский язык
    • адрес места проживания за границей

    Для юридических лиц РФ:

    • копия выписки из ЕГРЮЛ

    Для иностранных юридических лиц

    • выписка из Торгового реестра с нотариально удостоверенным переводом на русский язык
    • ФИО руководителя на русском языке
